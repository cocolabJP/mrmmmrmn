森見守者（もりみまもりもん）とは

IoTデバイス「りもん」は、森の様子を見守ります。 森を訪れる人やデバイスを見つけることが得意です。

「りもん」は、太陽の光で動きます。 なので、夜や雨の日は寝てしまいます。
